    Music Composition

    The score features a disco-inspired main theme built on retro synthesizers and layered string arrangements. Each musical element was carefully crafted to support the narrative while maintaining the nostalgic aesthetic.

    • Disco-inspired main theme with retro synthesizers
    • Character emotion mapping to harmonic progressions
    • Seamless integration with narrative pacing
    • Dynamic orchestration with layered strings
    • Memorable melodic hooks throughout

    Technical Approach

    Professional audio production with focus on cinematic depth and dynamic range optimization. The mix ensures clear separation while maintaining the warm, nostalgic character of the disco era.

    • Evolving textures without dialogue masking
    • Dynamic range optimization for animation
    • Retro aesthetic with modern production
    • Stereo field positioning for cinematic depth
    • Professional mixing and mastering

    • Replaced original composer from Ghent team

    • Multiple revision cycles for perfect sync
