Key Features

Everything You Need

Professional-grade automation for Unreal Engine developers

🔄

Total Rebuild Clean

Complete project refresh: removes Binaries, Intermediate, and Saved folders, regenerates Visual Studio solution, and compiles with build time tracking.

💾

Automated Backups

One-click project archiving with smart compression, timestamped archives, and selective backup of essential files (Source/Config/Content/Plugins).

📊

Disk Management

Detailed folder size breakdown with DDC cleaning that can free up gigabytes instantly. Smart formatting shows human-readable sizes with before/after stats.

🔍

Intelligent Detection

Auto-finds UE installations by scanning drives and checking Windows Registry. Supports multiple UE versions (4.x and 5.x) with fallback GUI folder picker.

📝

Professional Logging

Automatic documentation of every operation with timestamped files, auto-cleanup of logs older than 30 days, and detailed error reporting.

🎨

Modern Interface

Dark theme with custom borderless title bar, color-coded buttons for visual distinction, and real-time console feedback with green terminal text.
