Overview

Unreal Engine Rebuilder started as a simple solution to a common problem: repetitive project maintenance tasks that waste valuable development time. What began as a batch script evolved into a comprehensive C# application with a modern GUI, designed to streamline the entire Unreal Engine development workflow. I created this tool to make life easier for myself and fellow students working with Unreal Engine, and it has proven invaluable in day-to-day development.

5-10 Minutes Saved Per Build
5-50 GB Disk Space Reclaimed
100% Automation Coverage

Key Features

🔄

Total Rebuild Clean

Complete project refresh: removes Binaries, Intermediate, and Saved folders, regenerates Visual Studio solution, and compiles the project with build time tracking.

💾

Automated Backups

One-click project archiving with smart compression, timestamped archives, and selective backup of essential files (Source/Config/Content/Plugins).

📊

Disk Management

Detailed folder size breakdown with DDC cleaning that can free up gigabytes instantly. Smart formatting shows human-readable sizes with before/after stats.

🔍

Intelligent Detection

Auto-finds UE installations by scanning drives and checking Windows Registry. Supports multiple UE versions (4.x and 5.x) with fallback GUI folder picker.

📝

Professional Logging

Automatic documentation of every operation with timestamped files, auto-cleanup of logs older than 30 days, and detailed error reporting.

🎨

Modern Interface

Dark theme with custom borderless title bar, color-coded buttons for visual distinction, and real-time console feedback with green terminal text.

Technical Implementation

Built with C# and .NET, the application features a modern WPF interface with async/await patterns for non-blocking operations. The architecture includes smart engine detection, comprehensive error handling, and state management for reliable operation.

  • Monday: Disk Usage Analysis

Why I Built This

As a student working with Unreal Engine, I found myself repeatedly performing the same tedious tasks: cleaning build artifacts, regenerating project files, managing disk space. These operations were necessary but time-consuming, taking 5-10 minutes each time. I wanted to automate these workflows so I could focus on what matters: actual development.

The tool evolved from personal scripts into a full application that my peers could use too. Seeing other students benefit from it and save time on their projects made the development effort worthwhile. It's now an essential part of my Unreal Engine workflow, and I continue to improve it based on real-world usage and feedback.
