Automated Tests

Manual Mobile Tests

Resize your browser to mobile width (< 768px) or use DevTools device emulation, then verify:

✅ Mobile Menu Toggle

  • Hamburger button is visible
  • Desktop nav links are hidden
  • Button has 3 horizontal lines
  • Button is clickable

✅ Mobile Menu Open

  • Click hamburger button
  • Menu slides in from right
  • Dark overlay appears
  • All nav links visible
  • Language switcher visible

✅ Mobile Menu Close

  • Click overlay to close
  • Menu slides out
  • Overlay fades away
  • Click hamburger again to close

✅ Touch Interactions

  • Tap hamburger button
  • Tap menu links
  • Tap overlay
  • Swipe gestures work

Desktop View Tests

Resize your browser to desktop width (≥ 768px), then verify:

✅ Desktop Navigation

  • Nav links visible in header
  • Hamburger button hidden
  • Links in horizontal row
  • Hover effects work

✅ Language Switcher

  • Language buttons visible
  • Buttons are clickable
  • Active language highlighted
